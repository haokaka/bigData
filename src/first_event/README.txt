本算法只有一个python代码文件，直接运行即可。
运行之前请添加训练数据和测试数据的路径（即更改代码第16行的data_path）
训练数据包括d_train_20180102.csv，d_test_A_20180102.csv，d_answer_a_20180128.csv
测试数据包括d_test_B_20180128.csv




本算法的代码，整个流程分为：特征工程，训练模型，预测结果部分
1）特征工程
首先使用中位数对缺失值进行填充，然后再加入一些特征。这部分工作已经再之前做好，结果保存在train_feat.csv和test_feat.csv中
其次删除乙肝相关的五个特征，因为缺失值太多
然后对于训练数据和测试数据中的每一个特征，使用没有缺失值的样例作为训练集，缺失值样例作为测试集(这部分训练集和测试集都是使用train_feat.csv和test_feat.csv中的数据)，用已经调好参数的模型训练和预测。使用预测的结果填充缺失值
2）本算法采用GBDT模型，使用10折交叉验证
3）预测b榜的测试数据
